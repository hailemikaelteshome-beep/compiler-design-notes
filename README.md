## ðŸ“˜ About Compiler Design

Compiler Design is a core subject in Computer Science that focuses on the principles, techniques, and tools used to build compilers. A compiler translates source code written in a high-level language (such as C or Java) into machine code or intermediate code that a computer can execute.

This repository covers both **theoretical concepts** and **practical examples** to make learning easier and more effective.
